# StockWise Lite - Inventory Management System

StockWise Lite is a professional Flutter inventory management system for small businesses. It helps users manage products, suppliers, sales, low-stock alerts, stock movement history, revenue, and profit through a clean dashboard interface.

## Main Features

- User account creation and login
- Clean professional dark dashboard
- Add, edit, delete, and search products
- Product categories
- SKU code generation
- Cost price and selling price tracking
- Automatic profit calculation
- Supplier management
- Sales recording
- Automatic stock reduction after sales
- Sale receipt popup
- Low-stock alerts
- Stock movement history / audit trail
- Reports page
- Analytics dashboard with category stock overview
- Local storage using SharedPreferences
